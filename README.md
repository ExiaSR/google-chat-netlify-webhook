# hangouts-chat-netlify-webhook

Deployment alert on [Hangout Chat](https://gsuite.google.com/products/chat/). Running on [Now](https://now.sh)

## Deploy

`$ now secret add hangout-chat-webhook-url <hangout_chat_webhook_url>`

`$ now`

`$ now alias <url>`
